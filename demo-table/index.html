Table Demo

A table is a way of creating a structured grid display of data. the tags we will be interested in are:

  • <table> : Identifies the table
  • <thead> : Identifies the header row of the table
  • <th> : Identifies the text in the head of the column
  • <tbody> : Identifies the body of the table (detail rows)
  • <tr> : Identifies the start of a body row
  • <td> Identifies a data cell on the body row
  • <tfoot> : Identifies the footer row of the table

all tags are paired tags The border parameter on the table tag will allow you to specify a border around your table. the colspan and rowspan parameter are use on the <td tab to allow you to merger columns on a single row and colums cells across rows.
